디자인 토큰

모든 시각 결정의 단일 진실 공급원. 3-tier 구조로 다크 모드를 자동 지원합니다.

토큰 시스템 구조

Primitives → Themes → Component의 3-tier. 컴포넌트는 항상 시멘틱(Themes) 토큰을 사용합니다.

Tier 1 Primitives — brand-700: #283dc5
↓
Tier 2 Themes — bg-brand → brand-700 (Light) / brand-600 (Dark)
↓
Tier 3 Component — Button uses var(--bg-brand)
테마 토글을 눌러보세요 — Tier 3 박스 색이 시멘틱 토큰을 통해 자동 전환됩니다

시멘틱 토큰 — Background / Text / Border

컴포넌트는 이 시멘틱 토큰만 사용합니다. 테마를 전환해 보세요 — 같은 토큰이 다른 원시 값에 매핑됩니다.

다크 모드 색조 함정 — 다크 배경에 같은 hue 텍스트를 올리면 색조 동조 효과로 묻혀 보입니다. Selected/Active 상태처럼 강조가 필요한 곳은 흰색 텍스트 + 채도 높은 배경을 사용하세요. 상세 가이드는 컴포넌트 → Selected/Active 상태 또는 02-design-principles.md 참조.
