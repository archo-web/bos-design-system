Version 4.0

BOS Design System

Flowbite + Tailwind 기반의 광범위한 디자인 시스템. 시멘틱 토큰, 다크 모드 정식 지원, 47개 컴포넌트 카테고리를 통해 일관된 사용자 경험을 빠르게 구축합니다.

탐색하기

디자인 시스템은 4개 영역으로 구성됩니다. 각 영역은 코드와 라이브 컴포넌트를 함께 제공합니다.

디자인 토큰

색상, 타이포, 간격, 그림자 — 시멘틱 토큰 489개로 다크 모드 자동 대응

컴포넌트

Button, Input, Modal 등 핵심 10개 컴포넌트의 라이브 데모와 코드 예시

레이아웃 패턴

List, Detail, Form, Dashboard, Auth — 5가지 페이지 패턴으로 95% 화면 커버

문서 곧 출시

디자인 원칙, 인터랙션 표준, UX 라이팅 가이드

이런 점이 좋아요

시멘틱 토큰

의미 기반 토큰(text-heading, bg-brand)으로 다크 모드 자동 대응

속성 조합형 컴포넌트

Button 하나에 500 variants — Color × Size × State 매트릭스로 모든 케이스 표현

다크 모드 정식 지원

Light/Dark 모드 변수 매핑이 내장 — 컴포넌트는 변경 없이 자동 전환

광범위한 컴포넌트

47개 카테고리, 수천 개 variant — 업무용/마케팅/모바일 모든 컨텍스트 커버

시작하기

코드에서 사용하려면 — 프로젝트의 CSS에 토큰 파일을 import 하세요.
@import url("./tokens/bos4-design-tokens.css"); /* 컴포넌트는 시멘틱 토큰을 사용합니다 */ .my-button { background: var(--colors-background-bg-brand); color: var(--colors-text-text-white); padding: var(--spacing-2_5) var(--spacing-4); border-radius: var(--border-radius-rounded-base); font-family: var(--typography-font-family-base); }
기본 사용법 — Light/Dark 모드는 html[data-theme]에서 자동 전환
